Siddhartha - Regular User

Primary User Stories

1 As a user, I want to sign up and log in easily, so I can access the platform.
2 As a user, I want to view a list of cakes with images and basic details, so I can explore options.
3 As a user, I want to open a cake detail page, so I can read full description and reviews.
4 As a user, I want to write a review, so I can share my experience.
5 As a user, I want to rate a cake, so others can quickly understand its quality.
6 As a user, I want to like/unlike a cake, so I can show my appreciation.
7 As a user, I want to search cakes, so I can quickly find specific cakes.
8 As a user, I want to view top rated / most liked cakes, so I can easily pick the best options.
